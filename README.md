Основна мета проєкту — надати користувачеві можливість зручно дізнаватися харчову цінність продуктів та вести персональний щоденник споживання їжі за день. Проєкт реалізовано на React, що забезпечує швидку та інтерактивну взаємодію з інтерфейсом.
На даному етапі я працюю над інтеграцією зовнішнього API для динамічного пошуку продуктів у реальному часі.
